# [HowTragic.net](http://howtragic.net "HowTragic.net")
Simple one page applet to calculate the size of a tragedy in the number of Xboxs that its worth. This is a joke project based on the Well There's Your Problem podcast.

## Tech Details
Written using TypeScript and Preact with Webpack as the bundler. This project was mostly an excuse to learn how to use these tools; especially TypeScript.

### Build and Run
Running this project is simple and can be dome with the following commands.
1. `git clone https://github.com/19dufbri/HowTragic.net.git`
2. `cd howtragic.net`
3. `npm i`
4. `npm run build` or `npm run serve` for the dev server
